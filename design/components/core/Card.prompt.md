Surface for grouped content. Default is white-on-paper, soft warm shadow. Variants — `flat` (hairline), `kraft` (paper-200), `bordered`, `pine` (inverted brand).

```jsx
<Card>
  <h3>Boomer's Diner</h3>
  <p>Two visits this week.</p>
</Card>
<Card variant="pine"><h3 style={{color:'var(--paper-100)'}}>Upper Delaware Passport</h3></Card>
```
